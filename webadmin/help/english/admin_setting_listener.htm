Navigation:  Administration > Settings >

Listener

Here you can configure web administration's listener.

 

 

Use TLS/SSL for secure remote administration

If you want to use HTTPS for remote administration, please check on this option and choose an SSL certificate.

 

Listener Port

This web administration's listener port, default is 5466.

 

Listener Address

This web administration listener address, default "*" means the admin port listens on all available IPs.

If you change it into a single IP address, then the admin port will listen on that IP address only.

 

SSL Certificate

An SSL certificate is necessary when the option "Use TLS/SSL for secure remote administration" is checked on. You can manage your SSL certificates in "Server -> Settings -> SSL Certificate Manager".
