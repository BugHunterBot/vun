Miscellaneous

 

 

Enable Directory List Cache

This feature allows the server to temporarily keep the directory list for a specified amount of time. And it will accelerate the file listing speed.

 

Enable Making Thumbnail Images

This option makes the server generate thumbnail images for the web client.

 

Enable Online Text Edit

This feature allows editing text files when using the web client.

 

Disable ECDSA SSH host key

ECDSA SSH host key is enabled by default, you can disable it for the connections from some old SFTP clients.

 

Show real owner/group in the file listing (Linux/Mac)

This feature allows the real file's owner/group in the file listing. Note: it is only available on Linux or Mac OS.

 

Enable FTP/SFTP command 'CHMOD' (Linux/Mac)

Allow FTP command "SITE CHMOD" (or similar SFTP command) to change the permissions of a file/folder. Note: it is only available on Linux or Mac OS.

 

User can change file's modification time

Allow or disallow FTP/SFTP client to change the file modification time.

 

WebLink Folder

All the WebLink files are stored under "Data/_WEBLINK" by default, and you can also specify a folder (like a shared folder) to store them.

 

UploadLink Folder

All the UploadLink files are stored under "Data/_UPLOADLINK" by default, and you can also specify a folder (like a shared folder) to store them.

 

SQLite DB Folder

The SQLite database file "bookmark_db" is stored under the folder "Data" by default, and you can also specify a folder (like a shared folder) to store it.

 

Working threads number

You can set the working threads number as many as the number of CPU cores, and you need to restart the WingFTP service after changing it.

 

Maximum Download/Upload Speed

Limits the maximum download/upload speed for the entire server.
