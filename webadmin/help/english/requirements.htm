Requirements

To properly run Wing FTP Server on your server machine or computer, the following requirements are recommended:

 

Hardware

•2-Core x86 CPU with 1GB RAM

 

Software

•64-bit Operating Systems: Windows 7/8/10/11, Windows Server 2008/2012/2016/2019/2022, Linux (such as Ubuntu 12.0+, RHEL 7.0+), Mac OS 10.5+.
•Web browser for Administration & Web Client: Google Chrome, Firefox, Edge, Safari, etc.

 

 

 

 

How to upgrade?

 

•Download the latest version.
•Backup your data files to a safe place (copy the whole "/Data" directory).
•Stop the Wing FTP Server service.
•Install the new version without any change.
