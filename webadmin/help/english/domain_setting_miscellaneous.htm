Miscellaneous

 

 

Enable FXP

Allow FTP transfer from server to server directly.

 

Enable FTP command 'OPTS UTF8 ON'

You can disable it to improve Unicode compatibility when using Windows Explorer.

 

Enable FTP command 'AUTH TLS/SSL.'

Allow explicit FTP over TLS/SSL.

 

File timestamp using GMT time

When enabled, file listing uses GMT timestamp. If not checked, the local time will be used instead.

 

Use customized logo

You can specify a customized logo to be displayed in Web Client.

 

Enable HTTP welcome message

You can enable this option and define the welcome message, then logged in HTTP(S) sessions will see the message.

 

Transfer buffer size

Transfer buffer size used to transfer data (default is 65535). You can increase this value if you want the server to read/write less often from/to the local storage.

 

SSH (SFTP) banner

You can define your own SSH (SFTP) banner string here.

 

SSL Certificate

Select an SSL certificate for this domain. You can manage your SSL certificates in "Server -> Settings -> SSL Certificate Manager". An SSL certificate is used for the TLS/SSL connections (FTPS or HTTPS).

 

SSH Host Key

Select an SSH host key for this domain. You can manage your SSH host keys at "Server -> Settings -> SSH Host Key Manager". An SSH host key is used for the SFTP connections.

 

SMTP Configuration

Select an SMTP configuration for this domain. You can manage your SMTP configurations at "Server -> Settings -> SMTP Configuration Manager". An SMTP configuration is used for sending emails.

 

Enable MODE Z compression

FTP extension command "MODE Z" is used to compress data for transfer.

 

Default compression level

If FTP client does not specify the compression level, the server will use this default value.

 

Maximum and Minimum allowed compression level

The server only allows the compression level between the minimum value and maximum value.
